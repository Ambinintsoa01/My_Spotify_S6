# Black list
- Ajouter dans un fichier `.\blacklist.txt` 
    - La liste des chanteurs blacklistées 
    - La liste des genre blacklistées
- Appeler cette liste dans le `sender` et exclure les chansons des artistes / du genre de blacklist dans l'API. (Sans supprimer de ./mp3-inbox)  

# Inclure / exclure
- Dans le formulaire de création de playlist, ajouter une option : 
    - Inclure (par defaut lorsqu'on arrive sur la page)
        - Inclure les chansons correspondants aux critères selecionnées.
    - Exclure 
        - Exclure les chansons correspondants aux critères selecionnées.
